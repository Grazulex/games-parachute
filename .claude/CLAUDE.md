# Préférences et Instructions Claude pour le Projet Parachut

## Gestion des Tâches

**IMPORTANT**: Toujours utiliser **Backmark MCP** pour la gestion des tâches dans ce projet.

- Utiliser les fonctions MCP Backmark (`mcp__backmark__*`) au lieu de `TodoWrite`
- Créer les tâches avec `mcp__backmark__backmark_task_create`
- Lister les tâches avec `mcp__backmark__backmark_task_list`
- Mettre à jour les statuts avec `mcp__backmark__backmark_task_edit`
- Utiliser les plans AI avec `mcp__backmark__backmark_task_ai_plan`
- Ajouter des notes avec `mcp__backmark__backmark_task_ai_note`

## Projet

Ce projet vise à recréer le jeu **Parachute** de Game & Watch (1981) avec un design authentique des années 80.

## Style

- Design LCD authentique avec segments
- Boîtier beige Game & Watch
- Sprites géométriques simples
- Affichage 7-segments pour le score
